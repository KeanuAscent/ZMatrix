Uninstalling ZMatrix is as simple as exiting the program, then selecting the 'Uninstall ZMatrix' shortcut in the ZMatrix installation folder.

To begin the Uninstall process, you must exit the ZMatrix program (if it is running).  You may exit the program using the 'Exit' option in the right click context menu shown below:

The Right Click Context Menu

After you have exited ZMatrix, you may perform the actual uninstallation by selecting the 'Uninstall ZMatrix' shortcut from the ZMatrix folder in the 'Programs' section of the Start Menu.  Alternatively, you may uninstall ZMatrix by using the 'Add/Remove Programs' dialog in the Control Panel.

Uninstalling ZMatrix will try to remove all files/registry entries created during the install process, as well as all files/registry entries created during the use of the program.  The one exception is the configuration files.  ZMatrix is designed to keep seperate configuration files for each different Windows user who runs the program.  These seperate configuration files are not removed by an uninstall.  If you must remove these files (recommended for expert users only), they are located in the 'Application Data' folder for each individual user, under a '.ZMatrix' sub-folder.
