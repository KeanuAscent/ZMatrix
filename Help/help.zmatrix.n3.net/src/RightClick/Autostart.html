This submenu allows you to specify under what circumstances ZMatrix should be automatically started on startup/login.

Auto-Start Submenu

Auto-Start For All Users

This menu item controls whether ZMatrix will automatically start on startup/login regardless of which user is logged in.  In order to change this setting, you must have adminstrator or power user rights on the machine.

Auto-Start For Current User

This menu item controls whether ZMatrix will automatically start on startup/login for the currently logged in user only.  You do NOT need administrator or power user rights to change this setting.  This setting is independent of the 'Auto-Start For All Users' setting.
