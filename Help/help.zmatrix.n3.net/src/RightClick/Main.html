The root/main entries within the right click context menu for ZMatrix allow you to control various aspects of the program.

Root/Main Entries In The Right Click Context Menu

Configure

Selecting this menu entry will launch the configuration dialog for ZMatrix.  This is the default menu entry for the right click context menu, which means that double clicking on the ZMatrix system tray icon will have the same effect as right clicking the icon, and selecting this menu entry.

Refresh

Selecting this menu entry forces ZMatrix to refresh it's internal settings.  This is useful for when a change has happened on the desktop that ZMatrix hasn't detected, refreshing will frequently force ZMatrix to detect the change.

Auto-Start Options

This sub-menu allows you to adjust the auto-start abilities of ZMatrix.

Screen Saver Options

This sub-menu allows you to adjust the screen saver abilities of ZMatrix.

Pause

Checking this menu entry causes ZMatrix to stop rendering until a later time when you resume or "un-pause".   Unchecking this menu entry causes ZMatrix to resume rendering.  Note that pausing/resuming can also be done by a single left click of the ZMatrix icon in the system tray.

Hire The Developer

Selecting this menu entry launches an information dialog on the author for those in need of a software developer.

Donate

Selecting this menu entry launches your default browser to display the ZMatrix donation web page.

Help

Selecting this menu entry launches this help documentation.

About

Selecting this menu entry launches the information/credits dialog for ZMatrix.  This dialog is useful for determining what version of ZMatrix you have installed.

Exit

Selecting this menu entry exits ZMatrix.  If you are upgrading or uninstalling ZMatrix, you must exit the program first.
