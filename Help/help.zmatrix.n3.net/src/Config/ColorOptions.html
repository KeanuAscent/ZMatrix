The 'Color' and 'Special String Color' sections within the configuration dialog allow you to specify the colors used when rendering the matrix streams.

Color Configuration Options

Fade

This button takes you to a color selection dialog which allows you to choose the color used when rendering the faded trailing characters which are left behind a matrix stream.

Foreground

This button takes you to a color selection dialog which allows you to choose the color used when rendering the (generally) brighter leading character in a matrix stream.

Background

This button takes you to a color selection dialog which allows you to choose the color used when rendering the background behind both faded trailing, and bright leading matrix stream characters.  Note that this background color is generally not used if the background opacity is set to transparent.  This background color is common to both regular and special string segments.
