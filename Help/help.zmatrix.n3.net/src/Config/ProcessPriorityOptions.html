The 'Process Priority' options within the configuration dialog allow you to specify the amount of CPU time that ZMatrix will be allocated in comparison to other processes running on the same machine.

Process Priority Configuration Options

Process Priority

This set of radio buttons allow you to choose the priority of the ZMatrix process.  An idle priority is reccommended.  With idle priority, the ZMatrix process should only be allocated CPU time by the operating system if no other processes on the machine need the CPU.
