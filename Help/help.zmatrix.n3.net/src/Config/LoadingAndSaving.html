All of the settings that are controllable by the configuration dialog (as well as the 'Priority' setting which is controlled by the right click menu) can be loaded from/saved to .cfg files.  The .cfg file format is a basic INI file format, but may change in the future.  Configuration settings are normally loaded and saved automatically to a file specific to the current logged in Windows user, this file is typically located at:

C:\Documents And Settings\\\Application Data\.ZMatrix\ZMatrix.cfg

If this file is deleted, ZMatrix will replace it with the default settings the next time it starts.  Also, ZMatrix automatically saves screensaver specific settings to another file typically located at:

C:\Documents And Settings\\\Application Data\.ZMatrix\ZMatrixScreenSaver.cfg

But, sometimes it may be necessary to save and load a configuration to another location for later use, such as restoring old/different settings, sending your settings to other users, etc...  This can be done using the 'Load...' and 'Save...' menu items within the 'File' menu of the configuration dialog.

File Menu Of The Configuration Dialog

Saving

After selecting the 'Save...' menu item, you will be prompted to select a '.cfg' file to save the settings to.  Simply select a new or existing file you want to save to, and the settings will be written to that file.  If you want to change the default settings (so that all users have certain settings when they use ZMatrix for the first time), you may save to the default configuration file typically located at:

C:\Program Files\ZMatrix\default.cfg

Loading

After selecting the 'Load...' menu item, you will be prompted to select a '.cfg' file to load the settings from.  Simply select the file you want to load from, and the settings will be updated to reflect the configuration settings within the file.  If you want to load the default settings, you may load the default configuration file typically located at:

C:\Program Files\ZMatrix\default.cfg

Loading Defaults

To quickly load the default settings, you may use the 'Load Defaults' menu item.  It will load the settings from the default configuration file typically located at:

C:\Program Files\ZMatrix\default.cfg
