The 'Background' options within the configuration dialog allow you to specify if and how the matrix streams will be blended with an existing desktop background.

Background Configuration Options

Opacity

This set of radio buttons allow you to choose whether the desktop background is treated as an opaque or transparent layer.

Mode

This set of radio buttons allow you to choose whether the font will be blended with the existing background bitmap, or whether the background will be treated as a flat color.

Blending

This set of radio buttons allow you to choose what bitwise operation is used when blending with the background.   Because blending is only done when the background mode is 'Bitmap', these values are disabled if the current background mode is 'Color'.
