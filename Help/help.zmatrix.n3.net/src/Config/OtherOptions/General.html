The 'General' options tab contains miscelaneous options related to the way ZMatrix renders the matrix streams.

General Options Tab

Max Stream

This slider and edit field group allow you to specify what the maximum number of matrix streams are.

Speed Variance

This slider and edit field group allow you to specify how varied the stream speeds are.  A speed variance of zero means that all the matrix streams will travel at the same speed.  A speed variance of 5 means that streams will move at anywhere from 1x to 5x the base speed.

Special String Probability

This slider and edit field group allow you to specify the probability of a special string occurring within a stream.  The units are percent.  A special string probability of 0 means that no streams will show a special string sequence, a probability of 100 means that all streams will show a special string sequence.

Refresh Time

This slider and edit field group allow you to specify how long the program will wait (in milliseconds) before moving a matrix stream travelling at the base speed forward by one character.  Varying the refresh time effectively changes the base speed of matrix streams.  The smaller the refresh time, the faster the matrix streams move down the screen.
