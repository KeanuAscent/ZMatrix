The cleanup options tabs contain options related to the cleanup of the trail of matrix streams.  There are two kind of cleanup operations possible: randomized and monotonous.   Randomized cleanup removes elements of the trail randomly, but within a specified range.  Monotonous cleanup removes elements of the trail at a fixed distance from the start of the matrix stream.  Both randomized and monotonous cleanups can be independently enabled or disabled.

Randomized Cleanup Tab

Randomized Cleanup Leading

This slider and edit field group allow you to specify how far behind the start of the matrix stream randomized cleanup can begin.  A leading value of 10 means that the randomized cleanup will start 10 characters above the start of the matrix stream.

Randomized Cleanup Space Pad

This slider and edit field group allow you to specify how far behind the leading of the randomized cleanup that randomized cleanup can extend.  A space pad value of 30 means that the randomized cleanup can extend up to 30 characters behind the leading of the randomized cleanup.

Monotonous Cleanup Tab

Monotonous Cleanup Back Trace

This slider and edit field group allow you to specify how far behind the start of the matrix stream monotonous cleanup will take place.  A back trace value of 50 means that the monotonous cleanup will happen at 50 characters above the start of the matrix stream.
