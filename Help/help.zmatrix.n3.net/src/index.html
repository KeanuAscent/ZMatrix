ZMatrix is an animated desktop background which displays streaming characters in a style similar to what was used in the movie 'The Matrix'.  Furthermore, the program is capable of functioning as a Windows screensaver as well as desktop background.  The program was based off of source code for a screen saver found at www.elouai.com, and is entirely free.  Below are examples of ZMatrix running in background mode, and in screen saver mode...
