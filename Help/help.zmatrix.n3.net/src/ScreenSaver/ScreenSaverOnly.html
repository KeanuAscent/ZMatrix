Using ZMatrix as a screen saver only (not as an animated background) is done by ensuring that the auto-start features are disabled, and specifying ZMatrixSS as the current windows screen saver.

Step 1: Disabling Auto-Start

Right click on the ZMatrix icon in the system tray.  Select the 'Auto-Start Options' sub menu.   Ensure that all auto-start options are un-checked.  If any auto-start options are checked, click on them, and they should become un-checked.  If the 'Auto-Start For All Users' option doesn't become checked/un-checked when you click on it, you likely don't have the administrator or power user permissions required to change that setting, and should ask a person who has those permissions to make that change.

Step 2: Setting ZMatrix As The Screen Saver

Right click on the ZMatrix icon in the system tray.  Select the 'Screen Saver Options' sub menu.   Select the 'Set As Screen Saver' option within the sub menu.

Step 3: Exiting ZMatrix

Right click on the ZMatrix icon in the system tray.  Select the 'Exit' menu item.

After these steps, ZMatrix should no longer be running in background mode (and will not automatically run in background mode on startup), but will be launched when Windows goes into screen save mode.
