The ZMatrix visualization plugin for Winamp contains multiple modules.  The modules each have different visual effects.

VU Modulate

This module varies the color of the ZMatrix effect based on the amplitude variance in the segment of music currently playing.  Fast increases in volume or fast decreases in waveform data cause the effect to lean towards the peak color value, and even, unchanging waveform data will cause the effect to lean towards the baseline color.

Frequency Modulate

This module varies the color of the ZMatrix effect based on the frequency variance in the segment of music currently playing.  A high average frequency in the spectrum data causes the effect to lean towards the peak color value, and low average frequency in the spectrum data causes the effect to lean towards the baseline color.
