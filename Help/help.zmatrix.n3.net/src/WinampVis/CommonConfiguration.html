Most or all of the Winamp visualization modules use a common configuration dialog.  This dialog allows the user to specify the baseline and peak properties of the effect (color offset, color scale, etc..).

Winamp Visualization Common Configuration Dialog

Base Color Offset

This is the offset applied to the color as the effect approaches the baseline of its range.

Base Color Scale

This is the scale/weighting applied to the color as the effect approaches the baseline of its range.  The units are percent.  A scale of 100 means scale by 1.0, i.e.: no change.

Peak Color Offset

This is the offset applied to the color as the effect approaches the peak of its range.

Peak Color Scale

This is the scale/weighting applied to the color as the effect approaches the peak of its range.  The units are percent.  A scale of 100 means scale by 1.0, i.e.: no change.

Global Color Offset

This is the offset applied to the value of the effect before color is calculated.

Global Color Scale

This is the scale/weighting applied to the value of the effect before color is calculated.  The units are percent.  A scale of 100 means scale by 1.0, i.e.: no change.
